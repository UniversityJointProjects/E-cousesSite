Лабараторная работа номер 7


Для предложенного варианта заданий необходимо реализовать веб-сайт в среде Django, удовлетворяющий следующим требованиям:
  • Количество таблиц в БД - не менее 5. Таблицы должны быть спроектированы в соответствии с тремя нормальными формами.
  • Должны быть реализованы 3 типа связей: one-to-one, one-to-many, man-to-many.
  • Таблицы создаются с использованием моделей в Django и миграций.
  • Как минимум, для 1 сущности должны быть реализованы CRUD операции (Create, read, update, delete) + представление списка сущностей.
  • Остальные таблицы должны быть доступны для редактирования в административной части сайта.
  • Необходимые пять таблиц должны быть разработаны для моделирования предметной области, заданной вариантом. Необходимо придумать эти таблицы, не забывая про здравый смысл.
